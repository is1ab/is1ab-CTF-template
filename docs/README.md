# 文件目錄

> 這裡只放**日常開發需要的核心文件**。詳細的設定和進階指南請看 [Wiki](https://github.com/is1ab/is1ab-CTF-template/wiki)。

## 新手路線圖

```
QUICKSTART.md（30 分鐘，從 clone 到 PR）
    ↓
docs/challenge-types-guide.md（選對題目類型）
    ↓
docs/public-private-explained.md（理解安全設計）
    ↓
docs/quick-reference.md（常用指令速查）
```
